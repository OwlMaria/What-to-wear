Приложение, помогающее решить извечный вопрос "Что мне надеть?"
По введенным пользователем параметрам подбирается комплект (по умолчанию из пяти элементов) из базы данных.
Также предполагаются функции загрузки собственного гардероба, оценки предложенных комплектов, создание собственных, подбор не только по заданным параметрам, но и к отдельной вещи.
У пользователя будет "личный кабинет" + можно будет открыть каждую вещь из базы и посмотреть информацию о ней.
